# Vendas de Supermercado – Análise de Dados

## Visão Geral

Este projeto analisa dados históricos de vendas de um supermercado, coletados ao longo de três meses em três filiais diferentes.

## Motivação

O rápido crescimento dos supermercados em cidades densamente povoadas aumentou o nível de concorrência. Esta análise explora o comportamento de vendas e os padrões dos clientes.

## Conjunto de Dados

- 3 filiais
- 3 meses de dados
- Múltiplos atributos relacionados a clientes, produtos e formas de pagamento

## Bibliotecas Utilizadas

- NumPy
- Pandas
- Seaborn
- Matplotlib

## Objetivo

Análise exploratória e preditiva de dados.
